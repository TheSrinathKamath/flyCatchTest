# NodeJS Express Server

## Server setup

1. Import the Database from `database/blogs.sql`
2. Configure `.env` files with
  2.1   IP
  2.2.  PORT
  2.3.  DB_USERNAME
  2.4.  DB_PASSWORD
  2.5.  DB_NAME=blogs
3. Run `npm i`

## Running server

Run `npm run start` for development purposes.
Run `node index.js` for production.
